연동 예정 API — /api/v1/auth
🔐 /api/v1/auth 현재 모의 처리
POST /auth/kakao/callback
카카오 OAuth code 교환 → JWT 발급
body: {code} → {access_token, refresh_token, is_new_user}
🌐 PUBLIC
POST /auth/naver/callback
네이버 OAuth code 교환 → JWT 발급
body: {code, state} → {access_token, refresh_token, is_new_user}
🌐 PUBLIC
POST /auth/phone/send-otp
핸드폰 인증번호 발송 (SMS)
body: {phone_hash} → {session_id, expires_in: 180}
🔐 USER
POST /auth/phone/verify
OTP 검증 + 번호 인증 완료
body: {session_id, code} → {success, phone_verified: true}
🔐 USER
GET /auth/me
현재 로그인 사용자 정보 조회
→ {id, nickname, sido, sigungu, grade, phone_verified}
🔐 USER

🔑
JWT 전략
Access Token: 15분
Refresh Token: 30일
Access → 메모리 (XSS 방어)
Refresh → HttpOnly Cookie
만료 → /auth/refresh 자동 호출
로그아웃 → Redis 블랙리스트
📱
핸드폰 번호 보호

서버: 해시값만 DB 저장

역추적: 수학적으로 불가
